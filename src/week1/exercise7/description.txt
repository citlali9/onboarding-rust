# Problem

Use joylabs/onboarding-rust to work on your personal branch to solve the following problem.

Given a column title as appearing in an Excel sheet, return its corresponding column number.

## Acceptance Criteria

For example:

```
    A -> 1
    B -> 2
    C -> 3
    ...
    Z -> 26
    AA -> 27
    AB -> 28 
    ...

```

Example 1:

Input: "A"
Output: 1
Example 2:

Input: "AB"
Output: 28
Example 3:

Input: "ZY"
Output: 701
