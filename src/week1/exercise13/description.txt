# Problem

Use joylabs/onboarding-rust to work on your personal branch to solve the following problem.

Write a function that takes a string as input and reverse only the vowels of a string.

## Acceptance Test
Example 1:

Input: "hello"
Output: "holle"
Example 2:

Input: "leetcode"
Output: "leotcede"

####Note:
The vowels does not include the letter "y".
